# 🏥 MedCare - Hospital Management System

## 📖 Project Overview
MedCare is a modern and responsive healthcare website developed as a student project.  
It provides users with essential medical information through a clean and user-friendly interface.

The project demonstrates front-end development skills and simulates a real hospital management system with an interactive design.

---

## 🎯 Project Objectives
- Create a professional healthcare website UI  
- Practice responsive web design  
- Improve skills in HTML, CSS (Tailwind), and JavaScript  
- Simulate a real-world medical system  

---

## ⚙️ How the Website Works

The MedCare website is a front-end application that runs directly in the browser.

### 🔹 Page Loading
The browser loads `index.html` along with styles and scripts.

### 🔹 Navigation
Users move between sections using the navigation bar.

### 🔹 Interactivity
JavaScript handles UI interactions such as buttons and form behavior.

### 🔹 Responsive Layout
Tailwind CSS ensures the layout adapts to all devices.

### 🔹 Static Content
All content is static and stored in project files (no backend).
